# パイロット結果の置き場

各パイロット(pilot_soa_visual / pilot_soa_visual2 / pilot_soa_audio / pilot.html ほか)の
完了画面で「結果JSONをダウンロード」したファイルを、このフォルダに入れてください。
Google Drive経由で解析側のMacに同期され、Claudeが読み取って確認します。

- ファイル名はそのままでOK。できれば「誰が・どの実験か」が分かるように
  例: pilot_soa_visual_kurihara_0714.json のように書き換えてもらえると助かります。
- あわせて主観メモ(疲れる/速すぎる/音が途切れる等)があれば、このフォルダに
  メモ.txt として置くか、チャットで伝えてください。
