# 記号体系と f_visual_karuta(t) の推定

本研究で扱う識別率曲線の命名を整理し、最終目標である「競技かるたの視覚版識別率曲線」をどう推定するかを記す。

## 0. 直感：同じ曲を別の楽器で（なぜ f_visual_karuta を測らなくてよいか）

**結論：競技かるたの視覚版カーブ f_visual_karuta(t) は直接測らなくてよい。** 本実験で測る「かな1文字の視覚・聴覚の識別率」だけから構成的に導ける。理由を楽器のたとえで説明する。

競技かるたの識別を「曲」だと思う。

- **曲のメロディ（どの音がいつ来れば、どの歌だと分かるか）** = 決まり字のロジック。耳で聴こうが目で読もうが**同じ**。「む」が来たら『むらさめの』、と決まるのは音でも文字でも変わらない。百人一首のリストだけで決まる、モダリティ非依存の部分。
- **各音の鳴り方（1音がどれだけハッキリ届くか）** = 1文字ごとの明瞭度。これだけが耳と目で違う（ピアノとバイオリンの音色差）。

ピアノの曲をバイオリンで再現したいとき、やるべきは「各音を同じ強さ・タイミングで鳴らす」ことだけ。各音さえ揃えればメロディ（曲全体）は自動的に同じように届く。曲全体を測り直す必要はない。

かるたに戻すと：

1. かるた識別＝「読みを1文字ずつ受け取り、どの歌か当てる」ゲーム。
2. 当てられる瞬間＝他の99首を排除できる文字が届いたとき（＝決まり字）。
3. 「あと何文字で一意に決まるか」は百人一首のリストだけで決まり、耳でも目でも同一。
4. だから耳かるたと目かるたで違うのは「各文字が今どれだけハッキリ届いているか」だけ。
5. その明瞭度→認識率を1文字について測るのが本実験（f_visual_kana と f_audio_kana）→ 変換 g。
6. g で字幕の各文字を音声と同じ明瞭度・タイミングで届くよう調整できる。
7. すると識別ゲームの進み方が完全に同じ → **f_visual_karuta(t) = f_audio_karuta(t) が自動成立**。

**文脈ブーストは怖くない**：「単独で測った認識率は歌の中ではもっと簡単では？」という心配があるが、その文脈の助けは耳と目で同じだけ働くので、両方いっしょに底上げされ等式は崩れない（ベイズ的には 認識＝信号品質×事前確率。g で信号品質を揃え、事前確率がモダリティ非依存なら結果も揃う）。

**唯一の確認ポイント**：音声に「文字以外の識別手掛かり（詠みの節回し・抑揚）」があれば、それは文字を揃えても視覚で再現できない。ただし競技かるたでは決まり字は音素で決まり、詠みの節は全歌ほぼ共通で識別情報を持たないので、この心配は実際にはほぼ無視できる。気になれば f_visual_karuta を丸ごと測るのではなく「音だけ／文字だけ／両方」で同一首を少数試す小チェックで足りる。

| 曲線 | 測る？ |
|---|---|
| f_visual_kana（目の1文字） | **測る**（本実験・視覚モード） |
| f_audio_kana（耳の1文字） | **測る**（本実験・聴覚モード） |
| f_audio_karuta（耳のかるた） | 既に Kikiwake で測定済み |
| **f_visual_karuta（目のかるた）** | **測らない**。上3つから構成的に決まる |

直接測定は「念のための検証」か「節回しの仮定が怪しい一部の歌」に限った最終手段に格下げできる。以下は同じ話を数式で精密化したもの。

## 1. 3つのデータと命名

実験を通じて、以下の識別率曲線が得られる。

| 記号 | 何の関数か | 入手元 | 軸の性質 |
|---|---|---|---|
| **f_audio_karuta(t)** | 100首から「どの歌か」を読み上げ音声で識別する確率 | Kikiwake（高尾友季・既存研究）で実測済み | **系列の蓄積**：読み上げ時刻 t が進み、決まり字に到達するほど識別できる |
| **f_visual_kana(p)** | コンテキストのない「かな」1文字を、視覚的に劣化させて識別する確率 | 本実験（視覚モード）で実測 | **1単位の明瞭度**：表示パラメータ |
| **f_audio_kana(frac)** | コンテキストのない「かな」1文字を、発話の frac% まで聞いて識別する確率（単音 時間ゲーティング/truncation） | 本実験（聴覚モード）で実測 | **1単位の部分供給**：発話をどこまで聞いたか |

> 旧称との対応: 旧 `f_audio(t)` = `f_audio_karuta(t)`、旧 `f(t)` = `f_visual_kana(p)`。
> 聴覚かな実験 `f_audio_kana` は v2（聴覚モダリティ追加）で新設。

`f_audio_karuta`（系列蓄積）と `f_*_kana`（1文字の明瞭度）は**直交した別の軸**にある点が最重要。単純比較はできない。

## 2. コンテキストなしかなのインクルーシブフォント

本実験後、`f_visual_kana` と `f_audio_kana` を対応づけることで、**文字レベルの変換関数 g** が得られる。

```
g = f_visual_kana^{-1} ∘ f_audio_kana
```

g は「ある聴覚明瞭度の1かな」を「同じ識別率を与える視覚明瞭度」に翻訳する。これが**コンテキストのないかなについて、視覚と聴覚を対応づけたインクルーシブフォント**の中身である（本研究の直接の成果物）。

## 3. f_visual_karuta(t) の推定（最終目標）

`f_audio_karuta(t)` に対応する視覚版 `f_visual_karuta(t)` を、**直接測定せずに**合成（推定）する枠組み。直接測定は最終手段として保留。

### 2階建てモデル

- **① 文字レベルの変換関数 g**（上記、本実験で実測）
- **② 系列レベルの識別関数 F**（コーパス由来・モダリティ非依存）
  「かな前置列のどこまで認識できたら100首のどれか一意に決まるか」という**決まり字の組み合わせ論**。百人一首コーパスから計算できる（cf. 松田孟留「決まり字の統計的分析」 https://www.jstage.jst.go.jp/article/jappstat/49/1/49_1/_article/-char/ja/ ）。耳で得たか目で得たかに依存しない純粋に言語的な関数。

### 合成式

時刻 t における識別は、その時点の各かなの認識プロファイルを F に通したもの：

```
f_visual_karuta(t) = F( { f_visual_kana(p_i(t)) }_i )
```

音声側は時刻 t で「既出かな＝ほぼ完全認識」＋「現在発話中のかな＝部分音声 f_audio_kana(τ)」というプロファイルを持つ。視覚字幕で**同じプロファイルを再現**するには、現在のかなの視覚明瞭度を `p = g(τ)` に設定すればよい。すると：

```
f_visual_karuta(t) = f_audio_karuta(t)   （構成的に一致）
```

つまり f_visual_karuta は別途推定するというより、**g で文字ごとの情報供給を一致させれば f_audio_karuta に一致するよう「構成」できる**。これがインクルーシブ字幕の設計目標そのもの。

### なぜ kana 実験が必要か

音声は1文字を**連続・部分的**に届ける（発話途中でも部分識別が進む）。その視覚的アナログが「部分的に明瞭な視覚かな」。`f_audio_kana(τ)`（τ＝そのかなの音声をどれだけ再生したか）↔ `f_visual_kana(p)` の対応 g が、音声の連続的な部分供給を視覚の連続的な明瞭度に変換する。これが無いと、音声の「文字単位の途中まで届く情報」を視覚で再現できない。

## 4. 等式が成り立つための仮定（＝研究で検証すべきズレ）

1. **F のモダリティ非依存性**：音声には韻律・抑揚（詠みの節回し）という視覚にない手掛かりがある。F が本当に同一か。
2. **孤立 vs 文脈の事前確率**：`f_*_kana` は**単独提示**で測る。一方 karuta は「100首のどれか」という強い top-down 事前確率がある。文脈ありの方が認識は易しいので、**合成は認識率を過小評価する（保守的）**。この補正量が実務上の核心。
3. **時間→かな対応・分節**：音声は連続・共調音、視覚字幕は離散。t↔かな の対応のとり方。

## 5. 実務レシピ

1. **検証**：Kikiwake を決まり字モデル F で分解し、`f_audio_karuta(t)` を再現できるか確認（サニティチェック）。
2. **合成**：各首の読み系列について g で音声明瞭度→視覚明瞭度を変換し、F に通して `f_visual_karuta(t)` を予測。
3. **補正**：文脈事前確率のブースト分を推定（必要なら kana を karuta 文脈内で少数実測）。
4. **最終手段**：合成と実測がズレる首だけ直接測定。

**要約**：g が「音↔光の文字単位トランスデューサ」、F が「コーパス由来の決まり字識別器」。
`f_visual_karuta = F ∘ g(音声 karuta ストリーム)` で推定でき、理想的には `f_audio_karuta` に一致する。
研究的価値は ① F のモダリティ非依存性の検証と ② 文脈事前確率の補正量の定量化にある。
