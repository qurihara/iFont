## 研究へのご協力のお願い

このたびは本研究へのご関心をありがとうございます。以下の説明をお読みいただき、内容にご同意いただける場合のみご参加ください。

### この研究の目的
この研究は、文字（ひらがな）が**どのくらい見えれば正しく読み取れるか**を調べるものです。競技かるたなどで、音声を聞き取りにくい方にも公平に情報が届く「字幕」のデザインを考えるための基礎データを集めています。

### 何をしていただくか
- 画面の中央に、ひらがな1文字が表示されます。はっきり見えることもあれば、ほとんど見えないこともあります。
- 表示された文字だと思うものを、画面の選択肢から**すばやく**選んでください（1問あたり5秒以内）。
- 確信が持てなくても、感覚で答えていただいて構いません。成績で評価することはありません。
- 練習5問のあと、本番200問にお答えいただきます。

### 所要時間
全体で**約20分**です。

### 取得するデータ
この研究では、次のデータのみを記録します。
- 各設問でお選びになった文字
- 各設問に回答するまでの時間（ミリ秒）
- 表示した文字や見えやすさなどの設定情報
- 参加識別のための情報（クラウドソーシングの worker_id、参加者ごとのランダムな完了コード）

### 個人情報について
- **お名前・ご住所・電話番号・メールアドレスなど、あなた個人を特定できる情報は一切収集しません。**
- 健康状態や障害の有無など、配慮を要する情報も収集しません。
- worker_id は、**報酬をお支払いするための照合にのみ**使用し、他の目的には使いません。

### 報酬と完了コードについて
- 実験を最後まで完了された方に、謝礼として **PayPay ポイント（1人あたり 360 円程度【要確認：金額】）**をお支払いします。
- 実験の最後に画面に表示される**完了コード**を、応募元（クラウドソーシング）の入力欄に貼り付けてください。これが報酬お支払いの対象となる確認になります。
- 完了コードは控えておくことをおすすめします（後述の問い合わせの際に必要になる場合があります）。

### 参加は任意です／いつでもやめられます
- 参加は**完全に任意**です。参加されなくても、あなたに不利益は一切ありません。
- 実験の途中でやめたくなった場合は、**いつでもブラウザを閉じて中断**できます。中断による不利益はありません（ただし、完了コードが出ないため、その回の報酬対象とならない場合があります）。
  - 【要確認：途中中断者への報酬の取り扱い方針があれば明記してください。】

### データの使われ方と保管
- 集めたデータは、研究室が管理する保管先（Google スプレッドシート）に記録し、研究目的（解析・学会発表・論文・学位論文など）にのみ使用します。
- 公表する際は、**統計的に集計した結果のみ**を用い、個人が特定される形で公表することはありません。
- データの保管期間：【記入欄：例）論文公表後 〇 年間】。期間経過後は適切に削除します。
- 本研究は GitHub Pages（実験ページの配信）、Google（データ記録）、Yahoo!クラウドソーシング／PayPay（募集・謝礼）といった外部サービスを利用します。これらのサービス上では各社の規約に基づいて情報が扱われますが、上記のとおり個人を特定する情報は収集しません。

### 同意の撤回について（重要）
- この実験は**無記名（匿名）**で行うため、いただいたデータとあなた個人を結びつける情報を研究者は保持していません。そのため、**参加後にあなたのデータだけを特定して削除することは原則できません。**
- どうしてもデータの削除をご希望の場合は、**手元の完了コード**を添えて下記の問い合わせ先にご連絡ください。完了コードからデータを特定できた場合に限り、削除に対応します。
  - 【要確認：完了コードによる事後削除の運用を行うかどうかを確定し、行わない場合はこの段落の記述を修正してください。】

### 対象となる方
- この実験は、**満〔18／20〕歳以上**の方を対象としています。【要確認：成年年齢の扱いを確定してください。】未成年の方はご参加いただけません。
