# 1文字 対 2文字の音響比較 — 共調音が弁別に効くかの網羅解析 (2026-07-03)

2文字課題をやる根拠は、前の文字の発音によって次の文字の発音が変わり（共調音）、弁別のしやすさに影響するはずだ、という考えである。全 72×72 の2文字音声が揃った今、2文字目（C2）が単体の1文字にくらべてどれだけ音響的に違うかを網羅的に測り、1文字課題だけで足りるのか、2文字課題が必要なのかを検証した。

## 方法

各音声の目標区間を切り出し、MFCC（13次）とその差分（13次）の区間平均をつないだ26次元の特徴量で表した。全トークンで標準化している。音高の交絡を避けるため、実験用の1文字（音高 B3）とは別に、2文字目と同じ音高 E4 の1文字プールを新たに合成し、主分析は E4 どうしで比べた。これにより、変わるのは前の文字だけになる。

再現するスクリプトは `acoustic_analysis/analyze_coarticulation.py`、結果は `acoustic_analysis/coarticulation_result.json` である。

## 発見1（最重要・要修正）: は と へ が助詞として読まれている

分析の副産物として、VOICEVOX が単独の「は」を /wa/、「へ」を /e/ と、助詞の読み方で合成していることが分かった。その結果、現在の刺激では「は」が「わ」と、「へ」が「え」と音響的に区別できなくなっている。これは意図しない読み方で、修正が必要な刺激生成のバグである。

原因は、ひらがなの「は」「へ」を単独でVOICEVOXに渡すと助詞として解釈されることにある。カタカナ（ハ、ヘ）で渡すと正しく /ha/、/he/ と読まれることを確認した。「を」はカタカナ（ヲ）でも /o/ のままで正しい。修正として、VOICEVOXへの問い合わせをカタカナに変換する処理を `build_2char_pool.py` と `build_1char_pool.py` に入れた。実験の前に、この修正でプールを作り直す必要がある。

## 発見2: 単独発話と語中発話は、同じ文字でも大きく違う

同じ文字を「単独で1文字」合成した場合と、「2文字目として語中で」合成した場合の音響を比べると、系統的に大きく違っていた。標準化した空間で、文字どうしの平均距離が5.60だったのに対し、単独発話から語中発話への系統的なずれ（全文字に共通する方向の差）は8.47と、文字間の差よりも大きかった。つまり、同じ文字でも、単独で言うか語中で言うかで、別の文字どうしよりも大きく音が変わる。

この系統的なずれを差し引いても、語中のC2が「自分自身の単独発話」に最も近く分類される割合は44%にとどまった。言いかえると、単独発話の1文字は、語中での発音を代表できていない。これは、1文字課題と2文字課題が、同じ目標文字について音響的に別物の刺激を提示していることを意味する。

## 発見3: 共調音（前の文字の影響）は本物で、濁音などで特に強い

音高をそろえた2文字の世界の中だけで見ると、語中のC2が「自分の平均」に最も近く分類される割合は、同音異字を除いて83.8%だった。裏を返すと、16%は前の文字の影響で別の文字の側に寄っている。共調音は無視できない大きさで存在する。

カテゴリごとに、共調音への強さ（文脈が変わっても自分らしさを保つ度合い）は次のように分かれた。

| カテゴリ | 共調音への強さ | 語中を単独発話で代表できる度合い |
|---|---|---|
| 撥音 ん | 100% | 64% |
| 無声摩擦 s（さすせそ） | 96% | 51% |
| 破擦・拗摩擦（し・ち・つ） | 94% | 54% |
| 無声破裂 t（たてと） | 90% | 26% |
| 無声破裂 p（ぱ行） | 89% | 64% |
| 鼻音 n（な行） | 88% | 51% |
| 無声破裂 k（か行） | 86% | 65% |
| 鼻音 m（ま行） | 84% | 25% |
| 流音 r（ら行） | 81% | 45% |
| 母音（あ行） | 71% | 18% |
| 有声破裂 b（ば行） | 70% | 54% |
| 有声摩擦・破擦 z（ざ行など） | 67% | 53% |
| 有声破裂 g（が行） | 66% | 40% |
| 有声破裂 d（だ行） | 65% | 45% |

前の文字の影響を最も強く受けるのは、有声破裂（濁音の が・だ・ば行）、有声摩擦（ざ行）、母音、半母音である。逆に、撥音「ん」、無声摩擦「さ行」、破擦「し・ち・つ」、無声破裂「か・た・ぱ行」は文脈が変わっても比較的自分らしさを保つ。ただし、これらの文脈に強い文字であっても、発見2の「単独と語中の系統的なずれ」は共通して受けている。

## 発見4: 本物の同音異字がある

「お」と「を」、「じ」と「ぢ」、「ず」と「づ」は、音響的にほぼ同一だった（標準化空間の距離がほぼ0）。これらは日本語として本当に同じ音なので、1文字課題でも2文字課題でも、どんな文脈でも音だけでは区別できない。弁別の下限として扱う必要がある（採点でひとまとめにする、あるいは出題から外すなど）。なお、発見1で述べた「は＝わ」「へ＝え」は、これとは違って合成バグなので、修正すれば区別できるようになる。

## 発見5: 実験用の1文字は音高が B3 で、さらに差が上乗せされる

参考として、同じ文字の B3 と E4 の距離（音高差だけの影響）を測ると、文字ごとにおよそ2.7から4.5で、共調音による広がり（平均3.03）と同じくらいの大きさだった。実験で使う1文字課題は音高が B3 なので、語中のC2（音高 E4）と比べると、発見2の「単独と語中の差」に加えて、この音高差も上乗せされる。

## 結論と方針

- 音響的な証拠は、**2文字課題をやるべきだ**という方向を強く支持する。語中の2文字目は、同じ文字を単独で言った1文字とは、系統的にも（単独と語中の差が文字間の差より大きい）、前の文字ごとの共調音でも（16%が別の文字に寄る）、大きく違う。1文字課題は語中の発音を代表できないので、置き換えにはならない。
- どの文字セットも「1文字だけでよい」とは言いにくい。あえて優先順位をつけるなら、共調音の影響が最も強い**濁音（が・だ・ば・ざ行）・母音・半母音で、2文字課題の必要性が最も高い**。撥音や無声子音は文脈に強いが、それでも単独と語中の系統的な差は受ける。
- 1文字課題は、発話の先頭（先行文脈なし、C1=∅）という別の条件として意味がある。2文字課題の代わりではなく、独立したベースラインとして残す位置づけになる。
- 本物の同音異字（お・を、じ・ぢ、ず・づ）は、音だけでは区別できないので、採点や出題で配慮する。
- 実験の前に、は・へ の合成バグを直してプールを作り直す。
