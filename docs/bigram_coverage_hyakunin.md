# 百人一首（上句）かな bigram カバー率調査 (2026-06-26)

next_phase_2char_audio.md の「再開時の最初の作業 1」。
**問い**: 競技かるた／kikiwake が実際に読み上げる「百人一首・上句」のかな2連接(bigram)は、
一般日本語のかな bigram をどれだけカバーするか。
→ ある程度カバーするなら、**kikiwake 実読みデータを使った2文字課題 (C1→C2) の射程**が広い、と判断できる。

## 方法

| 役割 | データ | 備考 |
|---|---|---|
| 百人一首（分子） | `Kikiwake/mfa_v3/modern_kana_kamigoku.json`（100首・上句・現代かな） | kikiwake が実読み・MFA境界を持つのは**上句**なので、2文字課題が供給しうる連接在庫と一致 |
| 一般日本語（分母） | Tatoeba 日本語文 **248,758 文**（CC-BY 2.0 FR）を `fugashi`+`UniDic-lite`(NINJAL) で読み(カタカナ)化→ひらがな化 | 公開辞書＋公開コーパス。完全再現可 |

- **単位**: 「かな2連接」の文字どおり **char（隣接かな2文字）を主指標**、mora（拗音=1モーラに結合）を従。
- **連鎖**: 各文/各歌の読みを連結し、ひらがな以外（漢字無読み・長音符ー・記号）で分断。語境界はまたぐ（連続読み相当）。上句は5-7-5の空白を除去し連続1チェーン扱い。
- **指標**:
  - `type coverage` = |HI∩Corpus| / |Corpus|（日本語 bigram **種**の何%をHIが持つか）
  - `token coverage` = Σ_{b∈HI} freq(b) / Σ_all freq（実テキストの bigram **出現**の何%がHIにもある連接か）★最重要
  - `reverse` = |HI∩Corpus| / |HI|（HI の連接が一般日本語で観測される率＝**妥当性**）

再現: `bigram_coverage/analyze_bigram_coverage.py`（venv: fugashi, unidic-lite, jaconv）。
コーパス: `bigram_coverage/jpn_sentences.tsv`（Tatoeba）。結果: `bigram_coverage/bigram_coverage_result.json`。

## 解釈（研究判断）

1. **HI の連接はほぼ全て現代日本語として妥当**（reverse 99.9%、一般コーパスに無い連接は `ゆぞ` の1種のみ）。
   → kikiwake 実読みの C1→C2 ペアは「古語特有の奇妙な連接」ではなく、**現代日本語の実在連接**。2文字課題の刺激として素直に使える。

2. **種(type)としては日本語 bigram の少数しか覆わない**（全かなで 12–17%、清音かるた集合でも 30%）。
   百人一首・上句は総モーラ約1,722・異なり連接852種にすぎず、日本語連接の**ロングテールは構造的に欠落**。

3. **出現(token)では約半分に届く**（全かな 41.5%、音声72字 45.2%、**かるた46字 52.8%**、最頻top100で52%）。
   → 頻出連接の「約半分」は kikiwake 実読みだけで練習・測定できる。残り半分が課題。

4. **取りこぼしは系統的で同定可能**。HI に無い高頻度連接の代表:
   - 促音 `った/って/かっ/いっ`、現代活用語尾 `てい/した/して/です/ます/んだ/んで`、
     漢語拗音 `じょ/しょ/ゅう/きょ/しゅ` … いずれも**現代語の形態（促音・拗音・口語活用）に集中**。
   - これらは古典和歌に出にくいだけで、日本語として基本的な連接。

## 含意 → 次フェーズ設計への接続

- **段階リリースの正当化**: kikiwake 実読みアームは「現代日本語連接の token で約半分・最頻でも半分」を**実音源で**カバーでき、生態妥当性が高い比較条件として価値がある。一方で**残り半分（特に促音・拗音・口語活用）は実読みでは原理的に埋まらない**。
- **歌声合成を主軸にする根拠が定量化された**: 全連接カバレッジ（特に欠落している現代語形態）は、韻律制御可能な合成で**狙って生成**する必要がある。
- **競技かるた優先スコープなら、まずは清音46字・52.8% token を実読みで押さえる2文字課題**から着手するのが費用対効果が高い（HI内 632連接が実在音源で利用可能）。
- 注意: token coverage は単位/集合で 41–53% と振れる。長音符ー（外来語）と漢字無読み区間は分断扱いのため、外来語特有連接は対象外。NTT『日本語の語彙特性』等のモーラ連接頻度を分母にすれば値は変動しうる（Tatoeba は口語寄り）。
