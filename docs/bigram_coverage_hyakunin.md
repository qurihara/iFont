# 百人一首（上句）のかな2連接 カバー率調査 (2026-06-26)

これは `next_phase_2char_audio.md` の「再開時の最初の作業 1」にあたる調査である。

この文書では、かな2文字の連なり（2つのかなが隣り合って出てくる組み合わせ。以下では「かな2連接」と呼ぶ）を分析の単位とする。

**問い**: 競技かるたや Kikiwake が実際に読み上げる「百人一首の上句」に含まれるかな2連接は、一般の日本語に現れるかな2連接をどれだけカバーしているのか。これがある程度カバーできているなら、Kikiwake の実際の読み上げ音声を使った2文字課題（先行文脈の1文字目から、ターゲットの2文字目へ進む課題）が、一般の日本語に対しても広い射程を持つと判断できる。

## 方法

| 役割 | データ | 備考 |
|---|---|---|
| 百人一首（分子） | `Kikiwake/mfa_v3/modern_kana_kamigoku.json`（100首・上句・現代かな） | Kikiwake が実際に読み上げ、Montreal Forced Aligner による境界情報を持つのは上句なので、2文字課題が供給しうる連接の在庫と一致する |
| 一般日本語（分母） | Tatoeba 日本語文 **248,758 文**（CC-BY 2.0 FR）を、`fugashi` と公開辞書 `UniDic-lite`（国立国語研究所）で読み（カタカナ）に変換し、ひらがなに直したもの | いずれも公開された辞書とコーパスであり、完全に再現できる |

- **分析の単位**: 「かな2連接」という言葉どおり、隣り合うかな2文字を主たる単位とした。あわせて、拗音（「ゃ・ゅ・ょ」など）を直前のかなと結合して1モーラと数えるモーラ単位でも算出し、こちらは従とした。
- **連なりの作り方**: 各文・各歌の読みを連結したうえで、ひらがな以外の文字（漢字で読みがつかなかった区間、長音符「ー」、記号）のところで連なりを分断した。語と語の境目はまたいで連続的に数えており、これは実際の連続した読み上げに対応する。上句については、五・七・五の区切りにあたる空白を取り除き、ひとつながりの連なりとして扱った。
- **指標**: 次の3つを用いた。
  - 出現の被覆率（token coverage）。一般日本語のテキストに現れたかな2連接の延べ出現数のうち、百人一首の上句にも含まれる連接が占める割合である。最も重視する指標である。
  - 種類の被覆率（type coverage）。一般日本語に現れたかな2連接の種類数のうち、百人一首の上句が含んでいる種類数の割合である。
  - 逆方向の被覆率。百人一首の上句に含まれる連接のうち、一般日本語でも観測される割合であり、百人一首の連接が現代日本語として妥当かどうかを表す。

再現する手順は次のとおりである。解析スクリプトは `bigram_coverage/analyze_bigram_coverage.py` で、`fugashi`・`unidic-lite`・`jaconv` を入れた仮想環境で動かす。コーパスは `bigram_coverage/jpn_sentences.tsv`（Tatoeba）で、結果は `bigram_coverage/bigram_coverage_result.json` に出力される。

## 解釈

1. 百人一首の上句に含まれる連接は、ほぼ全てが現代日本語としても妥当である。逆方向の被覆率は99.9%で、一般コーパスに現れない連接は「ゆぞ」の1種類だけであった。したがって、Kikiwake の実際の読み上げから取り出した2文字のペアは、古語特有の奇妙な連接ではなく、現代日本語にも実在する連接である。2文字課題の刺激としてそのまま使うことができる。

2. ただし、連接の種類という観点では、一般日本語のごく一部しか覆っていない。全てのかなを対象とすると11〜17%、清音だけの競技かるた集合に絞っても30%にとどまる。百人一首の上句は総モーラ数が約1,722で、異なる連接は852種類にすぎないため、日本語の連接の長い裾野は構造的に欠落している。

3. 一方、出現の頻度で見ると、およそ半分には届いている。全てのかなで41.5%、音声の72字集合で45.2%、競技かるたの46字集合では52.8%であり、最頻の上位100種類でも52%を捕捉している。つまり、頻繁に出てくる連接のおよそ半分は、Kikiwake の実際の読み上げだけで練習・測定できる。残りの半分が課題として残る。

4. 取りこぼしには明確な偏りがあり、原因を特定できる。百人一首の上句に含まれない高頻度の連接の代表例は次のとおりである。促音を含むもの（「った」「って」「かっ」「いっ」）、現代語の活用語尾（「てい」「した」「して」「です」「ます」「んだ」「んで」）、漢語に多い拗音（「じょ」「しょ」「ゅう」「きょ」「しゅ」）である。これらはいずれも、促音・拗音・口語的な活用といった現代語の語形に集中している。古典和歌には出てきにくいだけで、日本語としては基本的な連接である。

## 含意（次のフェーズの設計との関係）

- 段階的に実施していく方針を裏づける結果である。Kikiwake の実際の読み上げを使うアーム（条件）は、現代日本語の連接のうち、出現頻度で見て約半分、最頻のものでも半分を、実際の音源でカバーできる。生態学的な妥当性が高い比較条件として価値がある。ただし、残りの半分、とりわけ促音・拗音・口語的な活用は、実際の読み上げだけでは原理的に埋まらない。
- 歌声合成を主軸にする根拠が、数値として裏づけられた。欠落している現代語の語形を含めて全ての連接をカバーするには、韻律を制御できる歌声合成によって、狙って生成する必要がある。
- 競技かるたを優先するスコープであれば、まずは清音46字・出現の被覆率52.8%を実際の読み上げで押さえる2文字課題から着手するのが、費用対効果が高い。この範囲では、百人一首の上句に含まれる632種類の連接を、実在する音源としてそのまま利用できる。
- 注意点を述べる。出現の被覆率は、分析の単位や文字セットによって41〜53%の幅で動く。長音符「ー」を含む外来語の連接と、漢字で読みがつかなかった区間は、連なりを分断する扱いにしているため、外来語に特有の連接は対象外である。また、分母として、たとえば NTT の『日本語の語彙特性』のようなモーラ連接の頻度表を使えば、値は変わりうる。今回用いた Tatoeba は口語寄りのコーパスである点にも留意したい。
