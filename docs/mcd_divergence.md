# 単音と流暢発音の乖離を MCD＋DTW で網羅計算 (2026-07-03)

単音だけ路線を主軸にするなら、各かなを単独で発話した「単音」を、そのまま配って使うことになる。そこで、単音が流暢な発話（語中での発音）からどれだけ離れているかを、音響的に網羅して測った。明瞭性のサーベイ（docs/intelligibility_metrics_survey.md）が乖離の主指標として推奨した、メル・ケプストラム歪み（MCD）を動的時間伸縮（DTW）で整列させて測る方法を用いた。これは、これまで使っていた MFCC 距離を、標準的で引用しやすい指標に置き換えたものである。

## 方法

比較する2つの音は、次のように用意した。単音の側は、2文字目と音高をそろえるために、E4（329.63ヘルツ）で合成した1文字プールを新しく作って使った。語中の側は、2文字プールの2文字目である。どちらもモーラの本体（子音と母音の区間）を切り出し、pymcd の DTW つき MCD で距離を計算した。音高をそろえてあるので、この距離は主に音色（スペクトル包絡）の違いを表す。

分子となる比較は、全 72 字の2文字プールの2文字目 5,184 対すべてである。1つのかなにつき、そのかなの単音と、そのかなが2文字目にくる 72 通りの語中発音との距離を、網羅して求めた。

尺度の基準として、次の2つも計算した。ひとつは、別のかなどうしの単音の距離である（決定的に散らした 432 対）。もうひとつは、同じ単音を自分自身と比べた距離で、これはほぼ 0 になるはずのサニティチェックである。

## 結果1: 単音と語中の乖離は、別のかなどうしの違いより大きい

単音と語中の乖離は、平均 15.5、中央値 15.9 だった。一方、別のかなどうしの単音の距離は、平均 8.2、中央値 7.9 である。自分自身との距離は 0.0 だった。

つまり、同じかなの「単音」と「語中」の違いは、別々の2つのかなの単音どうしの違いの、およそ 2 倍ある。単音は、語中での発音を代表できていない。これは、以前の共調音の解析（docs/coarticulation_analysis.md の発見2）が MFCC 距離で示したこと、すなわち単独発話と語中発話の系統的なずれが文字間の差より大きいことを、標準的な MCD でも裏づけた。

## 結果2: 乖離は音素の種類で大きく違う

2文字目のかなを調音の種類で分けて集計すると、乖離の大きさは次の順だった。

| 種類 | 例 | 乖離の平均(MCD) |
| --- | --- | --- |
| 撥音 | ん | 22.9 |
| 母音 | あ い う え お | 20.7 |
| 半母音 | や ゆ よ わ | 18.4 |
| 有声破裂 | が だ ば行 | 17.6 |
| 有声摩擦 | ざ行 | 16.3 |
| 無声摩擦 | さ行 は行 | 16.1 |
| 無声破裂 | か た ぱ行 | 9.3 |
| 破擦（無声） | ち つ | 7.4 |

撥音、母音、半母音、それに濁音などの有声音では、単音と語中の乖離が大きい。逆に、無声の破裂音（か行・た行・ぱ行）と破擦音（ち・つ）では乖離が小さく、単音が語中の発音に近い。かな別で乖離が大きい順に並べると、上位は あ・ん・う・わ・え・で・が であり、小さい順では き・つ・く・ぽ・ぴ が並ぶ。

これは直観と合う。無声の破裂音や破擦音は、識別の手掛かりが子音の破裂や摩擦の部分に集中していて、その部分は前後の文脈で変わりにくい。だから単独でも語中でも似た音になる。一方、母音や有声音は、声帯の振動が続くあいだ前後の音と互いに溶け合うので、単独と語中で音色が大きく変わる。撥音「ん」は、後続がないと帰属先が定まらず、単独と語中で最も大きく違う。

## 注意点

MCD の絶対値は、文全体を比べる通常の使い方（4 から 8 くらい）より高く出ている。これは、比べているのが 0.2 秒ほどの単一モーラで、しかも単独発話と語中発話という性質の違う音どうしだからである。したがって、意味があるのは絶対値そのものではなく、基準との相対関係である。すなわち、自分自身との距離が 0、別のかなどうしが約 8、単音と語中が約 15.5 という並びが、乖離の大きさを示している。

なお、音高は単音側も E4 にそろえてあるので、音高差による見かけの乖離は除いてある。純粋に音色の違いを測っている。

## 結論

単音だけ路線を採ると、流暢な発話に対しておよそ「別のかなに変わるくらいの違いの 2 倍」の乖離を、音色の面で受け入れることになる。その乖離は音素の種類で偏っていて、母音・半母音・撥音・有声音で大きく、無声の破裂音と破擦音で小さい。

この結果は、単音だけ路線をやめる理由にはならない。単音だけ路線の狙いは、前の文字の影響を受けない一貫した発音を、利用者が1文字1対応で覚えられるようにすることにある。ここで測った乖離は、その一貫性と引き換えに失う「流暢さ・自然さ」の大きさを、音素ごとに定量化したものである。実装や運用でとくに気をつけるべきなのは、乖離の大きい母音・半母音・撥音・有声音だと分かった。共調音の効果を見積もる2文字課題を、検証として残す価値も、あらためて数値で裏づけられた。
